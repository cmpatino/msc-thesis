\section{Using TAID with On-Policy Rollouts and RKL}

\section{The capacity gap appears clearly under naive off-policy distillation}

SFT on teacher completions performs poorly, especially for the 0.6B student, and larger teachers make things worse. This establishes the problem.

\section{On-policy distillation is a better teacher-student interface}

OPD substantially improves AIME over SFT for both students and preserves IFEval. This is the main positive result.

\section{Hard top-1 OPD is still information-limited}

Top-1 labels compress the teacher distribution too aggressively. Diagnostics show high top-1 agreement between 30B and 235B teachers, so the larger teacher’s advantage is partly hidden.


\section{Distributional top-K objectives partially address that bottleneck}

Bucketed RKL exposes more of the teacher distribution and works best for the 4B student with the 235B teacher. But K=10 is not enough to make the 235B teacher useful for the 0.6B student.

\section{Using an intermediate teacher is useful to address the gap. (?)}

\section{Privileged context is useful but not a general solution}

PC can improve top-1 OPD for the tiny student and can preserve GPQA in some settings, but PC+RKL does not currently improve the main AIME results. Therefore PC should be framed as a conditional teacher-side aid, not the core solution.
